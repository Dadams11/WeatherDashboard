# Weather Dashboard

## Description
The Weather Dashboard is a web application that allows users to search for weather conditions by city. It displays the current weather along with a five-day forecast, making it easy to plan ahead for travel, events, or daily activities.

This project was built using HTML, CSS, JavaScript, Bootstrap, and the OpenWeather API. It also uses localStorage so previously searched cities can be quickly accessed again.

## Features
- Search for weather by city name
- View current weather conditions
- View a 5-day weather forecast
- Save previous city searches in localStorage
- Click previously searched cities to view weather again

## Installation
No installation is required to use the deployed application.

To run locally:
1. Clone the repository
2. Open the project in VS Code
3. Launch `index.html` in your browser

## Usage
1. Enter a city name in the search field
2. Click the Search button
3. View the current weather and five-day forecast
4. Use the saved city buttons to quickly search previous cities again
